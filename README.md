# 員工多功能管理系統 (My Super App)

這是一個基於 **Go 語言** 與 **Supabase** 開發的後端系統，旨在整合公司內部的日常功能。

## 🚀 專案願景
透過自動化工具，簡化員工打卡、海報票選、留言互動及餐點訂購流程，並利用 Go 語言處理高效能的圖片壓縮，節省雲端儲存空間。

---

## 📂 資料夾結構說明

* **`main.go`**: 專案總入口，負責路由分發。
* **`auth/`**: 身分驗證模組（登入、註冊、JWT 驗證）。
* **`punch/`**: 打卡模組（紀錄打卡時間與位置）。
* **`voting/`**: 票選模組（海報上傳、一魚兩吃圖片處理、投票邏輯）。
* **`comments/`**: 互動模組（員工留言板）。
* **`meals/`**: 訂餐模組（選單顯示與下單）。
* **`database/`**: 資料庫中控中心（Supabase 連線邏輯）。
* **`web/`**: 前端靜態網頁檔案（HTML/CSS/JS）。

---

## 🛠️ 技術棧 (Tech Stack)

* **語言**: Go (Golang)
* **資料庫**: Supabase (PostgreSQL)
* **圖片存儲**: Supabase Storage
* **部署平台**: Render
* **版本管理**: GitHub

## 💡 開發備忘錄 (Tips)
* 圖片上傳前務必在 Go 後端進行壓縮，以節省 Supabase 1GB 的存儲限制。
* Render 免費版有休眠機制，每日早上 8:50 可手動或自動喚醒伺服器以確保員工使用順暢。
